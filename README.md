### Execution

```
  git clone https://github.com/666aalexx/Zunlocker.git
  
  cd Zunlocker/
  
  chmod +x zunlocker.py
  
  python3 zunlocker.py -z zipfile -w wordlist -oN zpassword.txt
```

### Installation
```
  chmod +x install.py

  python3 install.py
```
